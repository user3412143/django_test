Отправка сообщений

Для того, чтобы начать отправлять сообщение, нужно обратиться по пути: http://127.0.0.1:8000/mailing/send/1/, где 1 - это message_id.

Все сообщения хранятся в БД, таблица mailing_mailing

Если сообщение не было найдено, то будет возрат json, в котором указано, что ошибка: http://127.0.0.1:8000/mailing/send/4/

Отмена отправки сообщений

http://127.0.0.1:8000/mailing/cancel/4/

Отправка данных через post-запрос

import requests

url = 'http://127.0.0.1:8000/mailing/upload/'

json_data = {'phone_number': '74953100000', 'tag': 'post', 'timezone': 'UTC+3', 'operator_code': 'mts'}

r = requests.post(url, json_data)

result = r.json()

if result['status'] == 'success':

print('[+] A message add in DB')

else:

print('[-] An error occurated')
